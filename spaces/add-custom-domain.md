# Add a custom domain

## **1: Add the domain here**

https://github.com/bonustrack/snapshot-spaces/blob/master/spaces/domains.json

`domains.json`

```json
{
  "my.custom.url": "my-space-key"
}
```

## **2: Configure the DNS**

You will need to add this as CNAME in your domain DNS `snapshotpage.b-cdn.net`
